# Database of databases for biology

## General databases
- [KEGG](https://www.genome.jp/kegg/) - Kyoto Encyclopedia of Genes and Genomes

- [SWISS-MODEL](https://swissmodel.expasy.org/) - Homology Models of Proteins and Complexes
